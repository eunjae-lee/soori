## Setup

Add a script to your `package.json` like the following:

```js
"scripts": {
  "build:soori": "soori build"
}
```

Create a config file named `soori.config.js` in the root dir.

### vite

If your project is using vite, you don't have to run `soori build` manually. Soori provides a vite plugin.

```js title="vite.config.ts"
import { defineConfig } from 'vite';
import { soori } from 'soori/vite';

export default defineConfig({
  plugins: [soori()],
});
```
